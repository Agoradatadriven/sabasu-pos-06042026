SABASU NOODLE BAR — POINT OF SALE SYSTEM
========================================

FOLDER STRUCTURE
  SABASU-POS/
    index.html        <- open this file to run the system
    css/
      style.css       <- all styling
    js/
      app.js          <- all logic + database

HOW TO RUN
  1. Keep the folder together (do not move files out of css/ or js/).
  2. Double-click index.html to open it in any browser
     (Chrome, Edge, Safari) on desktop, tablet, or phone.

FEATURES
  - Big menu images for an attractive customer-facing display
  - Editable CASHIER NAME (top bar + payment screen) printed on every receipt
  - Dine In / Take Out (C/O) order types
  - Discounts: Senior 20%, PWD 20%, Custom %, Custom Peso
  - Payment with Cash / GCash / Card + automatic change
  - Printable receipts with order no., date/time, cashier
  - Sales history, daily totals, CSV export
  - Menu Manager: add/edit/delete items and upload photos

DATA
  All menu items, sales, the cashier name, and order numbers are saved
  in the browser's storage (acts as the database). Use the same browser
  on the same device. Use "Export CSV" regularly to back up your sales.
